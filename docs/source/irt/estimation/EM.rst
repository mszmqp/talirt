=================
EM算法
=================

得到了 :math:`\alpha` 的基础上，然后可以得到 :math:`\beta_m` 。

.. math::
    \beta_m = arg \ min \sum_{i=1}^N L(y_i,F_{m-1}(x_i) + \beta h(x_i;\alpha_m))
